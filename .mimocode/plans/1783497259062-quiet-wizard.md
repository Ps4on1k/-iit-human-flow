# Plan: Interviewer pills, voting in sobesedovanie, stylish timeline

## Задачи

### 1. Интервьюеры — pills (теги) с CRUD

**Сейчас:** Вкладка "Собеседования" имеет dropdown для добавления интервьюера (по одному), но нет отображения списка интервьюеров как pills и нет удаления.

**Нужно:**
- Отображать интервьюеров как pills (Ant Design Tag) с именами
- Мульти-селект dropdown для добавления (вместо одиночного Select)
- Крестик на pill для удаления (только для RECRUITER и ADMIN)
- Для остальных ролей — только просмотр без редактирования

**Бэкенд:**
- `packages/backend/src/modules/interviews/interviews.controller.ts` — добавить `DELETE /interviews/:id`
- `packages/backend/src/modules/interviews/interviews.service.ts` — добавить метод `remove(id)`

**Фронтенд:**
- `packages/frontend/src/pages/candidates/CandidateDetailPage.tsx` — секция интервьюеров:
  - Pills с именами интервьюеров + кнопка удаления для RECRUITER/ADMIN
  - Select mode="multiple" для добавления из списка пользователей
  - Для HIRING_MANAGER и HR — только чтение (без Select и крестиков)
- Импорт `useAuthStore` для проверки роли

### 2. Голосование — в раздел "Собеседование"

**Сейчас:** Голосование показывается в шапке карточки кандидата (buttons) и в вкладке "Собеседования" (список голосов).

**Нужно:**
- Убрать кнопки голосования из шапки карточки
- В шапке — показать итоги голосования (👍 За / 👎 Против) рядом с именем как pills
- Вкладка "Собеседования" — кнопка "Проголосовать" + список голосов
- Остальное не менять

**Фронтенд:**
- `CandidateDetailPage.tsx`:
  - Шапка: рядом с именем показать `votes.for` и `votes.against` как маленькие colored pills (без кнопки голосования)
  - Вкладка собеседований: кнопка "Проголосовать" + список голосов (оставить как есть)

### 3. Стильный таймлайн флоу

**Сейчас:** Timeline — прямоугольные pillы с `borderRadius: 2`.

**Нужно:**
- Скруглить углы: `borderRadius: 16` для pill'ов стадий
- Добавить стрелки/линии между стадиями со скруглением
- Цвет активной стадии ярче, прошедшие — мягче
- Hover-эффект на стадиях

**Фронтенд:**
- `CandidateDetailPage.tsx` — блок горизонтального таймлайна

3. `docker compose up -d` — контейнеры стартуют
4. Карточка кандидата: pills интервьюеров с крестиками (для RECRUITER/ADMIN)
5. Шапка: итоги голосования рядом с именем
6. Вкладка "Собеседования": кнопка голосования + список
7. Таймлайн: скруглённые pill'ы, hover-эффекты
