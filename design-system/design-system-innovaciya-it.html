01 — Логотип

Применение логотипа

Логотип существует в двух вариантах: горизонтальный (основной) и вертикальный. Минимальный размер — 80px по ширине. Охранная зона — размер символа × 0.5 с каждой стороны.

03 — Типографика

Шрифтовая система

Основной шрифт — Manrope. Все веса от Light (300) до ExtraBold (800). Альтернатива для заголовков — Geologica 700/800.

05 — Сетка

Макетная сетка

12-колоночная сетка. Гаттер 24px. Боковые поля 64px для desktop, 24px для mobile. Gap между блоками — 2px (фирменный приём бренда).

06 — Иконки

Система иконографики
